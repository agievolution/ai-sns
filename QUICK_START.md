## 📁 新架构概览

### 前端模块结构

```
renderer/js/
├── core/               # 核心层 - 路由、事件总线、存储
├── shared/             # 共享层 - Modal、Notification组件
└── modules/            # 业务层 - 6个独立模块
    ├── home/          # 首页（4个文件）
    ├── sns/           # 地图社交（6个文件）
    ├── agent/         # AI聊天（6个文件）
    ├── km/            # 知识库（4个文件）
    ├── tools/         # 工具市场（4个文件）
    └── web/           # LLM服务（4个文件）
```

### 后端模块结构

```
backend/
├── config/            # 配置层 - 设置、数据库
├── core/              # 核心层 - 依赖注入、认证
├── shared/            # 共享层 - WebSocket、AI客户端
├── database/          # 数据层 - 37个ORM模型 + 仓库
└── modules/           # 业务层 - 6个独立模块
    ├── agent/         # Agent管理（5个文件）
    ├── chat/          # 聊天服务（6个文件，含SSE）
    ├── map/           # 地图功能（6个文件，含WebSocket）
    ├── km/            # 知识库（5个文件）
    ├── system/        # 系统配置（5个文件）
    └── plugins/       # 插件管理（5个文件）
```
