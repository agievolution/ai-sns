🧪 SNS Profile 功能测试

说明：此页面用于测试 SNS Profile 页签的打开和关闭功能。
请确保后端服务器正在运行（http://localhost:8788）。

📖 功能说明

open_sns_profile(url) - 在右侧状态面板中打开一个新的 Profile 页签，显示指定 URL 的内容

close_sns_profile() - 关闭 Profile 页签

🔧 测试操作

📝 预设测试 URL

💻 调用示例代码

// 打开 Profile 页签
open_sns_profile('https://www.ai-sns.org');

// 关闭 Profile 页签
close_sns_profile();

📋 实现细节

  • ✅ Profile 页签会添加到右侧状态面板的底部页签栏
  • ✅ 页签内容使用 iframe 加载指定的 URL
  • ✅ 页签标题显示 "Profile" 并带有关闭按钮 (×)
  • ✅ 点击关闭按钮或调用 close_sns_profile() 可关闭页签
  • ✅ 如果 Profile 页签已存在，会更新 URL 而不是创建新页签
  • ✅ 关闭 Profile 页签后会自动切换到第一个页签（Process）
